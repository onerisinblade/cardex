# Карточки к экзамену

Мини-приложение для подготовки по карточкам.

## Что есть

- Колода с таблицей интегралов.
- Создание своих текстовых колод.
- Добавление, редактирование, удаление и сортировка карточек.
- Перемешивание и порядок по умолчанию.
- Тёмная и светлая тема.
- Импорт карточек из `.txt` файла.
- Экспорт выбранной колоды в `.txt` файл.
- Данные каждого пользователя сохраняются локально в его браузере.

## Импорт карточек

В разделе **Управление** нажми **Импортировать** и загрузи `.txt` файл.

Формат файла:

```txt
Вопрос 1 | Ответ 1
Вопрос 2 | Ответ 2
Вопрос 3 | Ответ 3
```

Одна строка = одна карточка. Карточки добавляются в текущую выбранную колоду.

Для обычных колод это простой текст. Для колоды интегралов можно импортировать LaTeX в таком же формате.

## Экспорт карточек

В разделе **Управление** нажми **Экспортировать**. Приложение скачает выбранную колоду в `.txt` формате:

```txt
Вопрос | Ответ
```

Этот файл можно отправить себе на телефон или одногруппнику и потом импортировать в приложении.
